
# KeysKingdom UI Guide

## Design Tokens
- **Primary Gold**: `#fbbf24` (Amber-400)
- **Obsidian Background**: `#0c0a09` (Stone-950)
- **Arcane Blue**: `#22d3ee` (Cyan-400)
- **Magic Purple**: `#d946ef` (Fuchsia-400)

## Layout Rules
- **Mobile (< 768px)**: HUD items stack. Modals use 95% width. Tooltips are triggered by an "Info" toggle.
- **Desktop (> 1024px)**: HUD horizontal. Tooltips on hover. Side news scroll visible.

## Components
- **Glass Panel**: `bg-indigo-950/90 backdrop-blur-xl border-2 border-amber-600/50`
- **Action Button**: `bg-gradient-to-b from-amber-400 to-amber-700`
