
# Changelog

All notable changes to this project will be documented in this file.

## [3.0.0] - 2023-10-27
### Rebrand & UI Refactor
- **Branding**: Official transition to **KeysKingdom**.
- **UI Refactor**: Modernized Obsidian & Gold aesthetic across all interfaces.
- ** Keeper of Keys**: Advisor renamed and re-contextualized.
- **Chronicle of Keys**: Enhanced save/load manager with better visual feedback.
- **The Vault of Keys**: Updated research interface with jewel-toned nodes.
- **Grimoire Terminal**: Refined developer console with amber-gold styling.

## [2.1.0] - 2023-10-27
### Visual Overhaul
- **Procedural Buildings**: Enhanced `BuildingSystem` variations.
- **Lighting Engine**: Brightened ambient lighting.
- **UI Contrast**: "Midnight & Gold" theme introduced.

## [0.1.0] - 2023-10-27
### Added
- Initial repository baseline.
- Core simulation engine.
- 3D Isometric Map with R3F.
- AI integration via Google Gemini.
